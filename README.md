# 💻 BaseCamp 종합 학습 요약 (1~3회차)

프론트엔드 부트캠프 BaseCamp 1회차부터 3회차까지의 학습 내용을 종합적으로 요약했습니다.
이 과정은 웹 페이지의 **구조(HTML)**와 **스타일(CSS)**의 기본 원리를 이해하는 데 중점을 둡니다.

---

### 📂 요약 폴더 구조

BaseCamp/ 
├── 1회차/ # HTML 기본 및 컨테이너 │ 
|
├── 2회차/ # HTML 상호작용 및 구조화 (링크, 폼, 테이블) │ 
|
└── 3회차/ # CSS 기본 및 박스 모델, 스타일링 |


### 🎯 종합 학습 목표

* **웹 구조화 (HTML)**: 웹 페이지의 내용과 구조를 정확하게 정의할 수 있습니다.
* **사용자 상호작용**: 링크, 목록, 표, 폼(Form)을 통해 사용자 경험을 설계하고 데이터를 수집할 수 있습니다.
* **시각적 디자인 (CSS)**: 웹 요소에 스타일을 적용하고, **박스 모델**을 이해하여 레이아웃을 제어할 수 있습니다.

---

### 💡 회차별 주요 학습 내용 요약

#### 1️⃣ 1회차: HTML 기본 구조 및 컨테이너

| 주요 내용 | 핵심 태그 / 속성 | 설명 |
| :--- | :--- | :--- |
| **HTML 기본 구조** | `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>` | 웹 문서의 필수 기본 틀 구성. |
| **컨테이너 & 전역 속성** | `<div>`, `<span>`, `id`, `class`, `title` | 요소 그룹화 및 공통 속성 사용법 학습. |
| **이미지 삽입** | `<img>`, `src`, `alt` | 웹 페이지에 이미지 표시 및 경로 설정. |

#### 2️⃣ 2회차: HTML 상호작용 및 구조화

| 주요 내용 | 핵심 태그 / 속성 | 설명 |
| :--- | :--- | :--- |
| **링크** | `<a>`, `href`, `target` | 외부/내부 페이지 연결 및 새 창 열기. |
| **목록 (List)** | `<ul>`, `<ol>`, `<li>` | 순서 없는/있는 목록을 통한 정보 정리. |
| **표 (Table)** | `<table>`, `<tr>`, `<th>`, `<td>` | 데이터를 행과 열로 구조화. |
| **양식 (Form)** | `<form>`, `<input>`, `<label>`, `<select>` | 사용자 입력 필드 정의 및 데이터 전송 준비. |
| **문서 정보** | `<meta>` | 문자 인코딩, 뷰포트, SEO 관련 정보 설정. |

#### 3️⃣ 3회차: CSS 기본 및 박스 모델

| 주요 내용 | 핵심 개념 / 속성 | 설명 |
| :--- | :--- | :--- |
| **CSS 적용 방법** | Inline, Internal, External | 스타일을 HTML에 연결하는 3가지 방식 및 **우선순위** 이해. |
| **선택자 (Selector)** | `.class`, `#id`, `tag`, `*` | 원하는 요소에 스타일을 지정하는 방법. |
| **박스 모델** | `width`, `height`, `margin`, `padding`, `border` | 모든 요소는 박스라는 기본 개념 하에 크기와 여백 제어. | 
| **스타일링** | `background`, `color` (HEX, RGB), `font-*`, `text-align` | 배경, 색상, 폰트 등 시각적 효과 부여. |

### 🛠️ 최종 실습 결과물

1회차의 `profile.html`을 기반으로 3회차에서 배운 CSS를 적용하여 **`css_profile.html`**을 완성했습니다.

* `css_profile.html` (HTML): 구조와 내용
* `styles.css` (CSS): 중앙 정렬, 배경색, 폰트, 호버 효과 등 스타일링 적용

이 과정을 통해 프론트엔드 개발의 기초인 HTML과 CSS의 핵심을 탄탄하게 다질 수 있었습니다.
