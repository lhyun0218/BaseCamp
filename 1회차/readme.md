### 📚 주요 학습 내용 및 실습 코드 분석

#### 1. HTML 문서의 기본 구조

모든 HTML 문서는 다음 요소를 포함하는 기본 구조를 가집니다.

* `<!DOCTYPE html>`: 문서 유형 선언.
* `<html lang="en">`: HTML 문서의 시작. `lang`은 전역 속성입니다.
* `<head>`: 문서의 메타데이터 (제목, 문자 인코딩 등) 설정.
* `<body>`: 웹 브라우저 화면에 실제로 표시될 내용을 담는 곳.

#### 2. 컨테이너 태그 및 전역 속성 (`08.html` 분석)

##### 컨테이너 태그

| 태그 | 유형 | 설명 | `08.html` 사용 예시 |
| :--- | :--- | :--- | :--- |
| `<div>` | **블록 요소** | 내용 구분을 위한 **가장 일반적인 컨테이너** | `<div id="study">`, `<div id="diet">` |
| `<span>` | **인라인 요소** | 텍스트의 **특정 부분**에 스타일을 적용하거나 구별할 때 사용 | `<span title="...">` |

##### 전역 속성 (Global Attributes)

모든 HTML 태그에서 공통으로 사용할 수 있는 속성입니다.

* `id`: 문서 내에서 **유일**해야 하는 고유 식별자. (`id="study"`)
* `class`: 요소를 그룹화하여 **CSS 스타일**을 적용할 때 사용. (`class="text content"`)
* `title`: 요소에 대한 추가 정보 제공. 마우스를 올리면 툴팁으로 표시.

#### 3. 이미지 삽입 (`<img>` 태그) 및 경로 설정 (`09.html`, `profile.html` 분석)

`<img>` 태그는 웹 페이지에 이미지를 삽입하는 **빈 태그 (Empty Tag)** 입니다.

| 속성 | 설명 | `09.html` 사용 예시 |
| :--- | :--- | :--- |
| `src` | **이미지 파일의 경로** (Source) 지정. | `src="/images/daejeon.webp"` (루트 기준) |
| `alt` | 이미지를 표시할 수 없을 때 대신 보여줄 **대체 텍스트**. | `alt="대전의 랜드마크 한빛탑..."` |
| `width`, `height` | 이미지의 너비와 높이를 픽셀 단위로 지정. | `width="300"` |

> **경로 설정 (Path)**
> * `09.html`에서는 `src="/images/daejeon.webp"` (루트 디렉터리 `/`를 기준으로 함)
> * `profile.html`에서는 `src="images/navi.jpg"` (현재 파일의 상대 경로 `./` 생략)
>   * `profile.html`과 `images` 폴더가 같은 `1회차` 폴더 내에 있으므로 상대 경로를 사용합니다.

### 📝 추가 학습 요소 (`profile.html` 참고)

| 태그 | 설명 |
| :--- | :--- |
| `<h1>` ~ `<h6>` | 제목 태그. `<h1>`이 가장 중요한 제목입니다. |
| `<p>` | 단락(Paragraph)을 나타냅니다. |
| `<br>` | 강제 줄 바꿈 (Line Break). |
| `<hr>` | 수평선 (Thematic break)을 삽입합니다. |
| `<strong>` | 내용의 **중요성**을 강조합니다. |
| `<em>` | 내용의 **강조** (Emphasis)를 나타냅니다. |
| `<mark>` | 형광펜 효과처럼 **하이라이트**합니다. |
| `&copy;` | HTML 특수 문자 (Entity)로 **저작권 기호**를 표시합니다. |
