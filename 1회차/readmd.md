### 📚 주요 학습 내용 및 실습 코드 분석

#### 1. 링크 설정 (`10.html` 분석)

* **`<a>` 태그**: 하이퍼링크를 생성합니다.
    * `href`: 이동할 **URL 주소** 또는 **파일 경로**를 지정합니다.
    * `target`: 링크가 열릴 위치를 지정합니다.
        * `_self`: 현재 창에서 열기 (기본값)
        * `_blank`: 새 창이나 새 탭에서 열기
    * **경로 사용**: `href="../1회차/profile.html"`처럼 상대 경로를 사용하여 내부 파일로 이동할 수 있습니다.

#### 2. 목록 만들기 (`11.html` 분석)

| 태그 | 유형 | 설명 | 사용 속성 |
| :--- | :--- | :--- | :--- |
| `<ul>` | **순서가 없는 목록** (Unordered List) | 항목을 나열하며 기본적으로 **점(•)** 모양의 마커를 사용합니다. | - |
| `<ol>` | **순서가 있는 목록** (Ordered List) | 항목에 **숫자나 알파벳** 순서를 매깁니다. | `start="1"`, `type="1"` (숫자), `type="a"` (알파벳) |
| `<li>` | 목록의 **개별 항목** (List Item) | `<ul>` 또는 `<ol>` 태그 안에 사용됩니다. | - |

> *참고: `11.html`의 마지막 예시처럼 `<ul>` 안에 또 다른 `<ul>`을 사용하여 중첩 목록을 만들 수 있습니다.*

#### 3. 표(테이블) 만들기 (`12.html` 분석)

* **`<table>`**: 전체 표의 구조를 정의합니다.
* **`<caption>`**: 표의 **제목**을 정의합니다. `<table>` 바로 아래에 위치합니다.
* **`<tr>`**: 표의 **가로 행** (Table Row)을 정의합니다.
* **`<th>`**: 테이블의 **제목 셀** (Table Header). 주로 굵은 글씨로 표시됩니다.
* **`<td>`**: 테이블의 **데이터 셀** (Table Data). 일반 데이터를 표시합니다.

| 태그 | 역할 | 위치 |
| :--- | :--- | :--- |
| `<thead>` | 표의 **머리글** 그룹 (Header) | `<tr>` 그룹을 감싸 표의 맨 위에 위치 |
| `<tbody>` | 표의 **본문** 그룹 (Body) | `<tr>` 그룹을 감싸 실제 데이터를 포함 |
| `<tfoot>` | 표의 **바닥글** 그룹 (Footer) | `<tr>` 그룹을 감싸 표의 맨 아래(데이터 요약)에 위치 |

#### 4. 사용자 입력 양식 (폼, Form)

사용자로부터 정보를 입력받아 서버로 전송하기 위해 사용되는 요소들입니다.

* **`<form action="..." method="...">`**: 양식 전체를 정의하고, 데이터를 보낼 서버 주소(`action`)와 전송 방식(`method`: `get` 또는 `post`)을 지정합니다.
* **`<label for="...">`**: 입력 요소에 대한 레이블(설명)을 정의합니다. `for` 속성과 입력 요소의 `id`를 연결합니다.
* **`<fieldset>`**: 관련 있는 입력 요소들을 그룹화하며, `<legend>`로 그룹의 제목을 지정합니다.

##### 주요 입력 태그 분석 (`13.html`, `14.html`, `15.html`, `final_form.html`)

| 태그 | 설명 | `type` 속성 (or 태그명) | 특징 |
| :--- | :--- | :--- | :--- |
| `<input>` | **단일** 입력 필드 | `text`, `password`, `color`, `number`, `range`, `submit` 등 | `placeholder`로 힌트 제공 가능. |
| `<input>` | 선택 입력 필드 | `checkbox` (중복 선택 가능), `radio` (name 속성이 같으면 하나만 선택 가능) | - |
| `<textarea>` | **여러 줄의** 텍스트 입력 필드 | (태그명) | `rows`와 `cols`로 크기 지정. |
| `<select>` | 드롭다운 목록 생성 | (태그명) | `<option>` 태그를 사용해 항목 추가. `multiple`로 다중 선택 가능. |
| `<button>` | 클릭 가능한 버튼 | (태그명) | - |
| `<progress>` | 작업 진행 상태를 표시합니다. | (태그명) | `value`와 `max` 속성을 사용합니다. |

#### 5. 문서 정보 관리 (`16.html` 분석)

* **`<meta>` 태그**: 웹 브라우저나 검색 엔진에게 문서에 대한 정보를 제공합니다. `<head>` 안에 위치합니다.
    * `charset="UTF-8"`: 문자 인코딩 설정.
    * `name="viewport"`: 반응형 웹을 위한 뷰포트 설정.
    * `name="author"`: 작성자 정보.
    * `name="description"`: 웹 페이지에 대한 짧은 설명 (검색 엔진에 노출).
    * `name="keywords"`: 웹 페이지의 핵심 키워드 (검색 엔진 최적화).
